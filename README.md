# PepeBot

PepeBot is a discord bot tailored to a group of individuals at [Chalmers University of Technology](https://www.chalmers.se/en/) studying Computer Engineering.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install PepeBot.

```bash
npm install
```

## Usage
Start the bot with node.

```bash
node bot.js
```

## Contributing
Feature requests are welcome, but the project not meant to grow into something generic or distributable. It was created for a specific use case and will continue to receive updates for this specific use case without major consideration to scalability or adaptation.
